Unidades de medida para dados de armazenamento

Computadores (e outros dispositivos eletrônicos) trabalham com impulsos elétricos que, como forma de facilitar a nossa compreensão, são representados apenas por 1 ou 0, razão pela qual essa abordagem é chamada de sistema binário. Para o armazenamento de dados, são usados unidades decimais e binárias de medida para expressar o tamanho desses dados de armazenamento.

Unidades decimais como kilobyte (KB), megabyte (MB) e gigabyte (GB) são comumente usadas para expressar o tamanho dos dados. Unidades binárias de medida incluem kibibyte (KiB), mebibyte (MiB) e gibibyte (GiB).
